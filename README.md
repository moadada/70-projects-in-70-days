## Key Technical Learnings

- File extensions must be visible; Windows hidden extensions caused `index.html.html` errors. Solution: enabled "File name extensions" and used quoted filenames in Notepad.
- API free tiers expire or have balance limits. Switched from DeepSeek to Gemini API mid-build.
- OneDrive interferes with local file linking. Moved all projects to `C:\` for reliable development.
- Never hardcode API keys in public repositories. Use environment variables or local configuration files.
